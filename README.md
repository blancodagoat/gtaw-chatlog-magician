<h1 id="chatlog-magician-for-gta-world">Chatlog Magician for GTA World</h1>
<p>This is a <strong>remake</strong> of the <a href="https://github.com/ulasbayraktar/chatlog-magician">original Chatlog Magician</a> project, which converts SA-MP logs into images, making screenshot editing easier.</p>
<h2 id="features">Features</h2>
<ul>
<li><strong>Modern Design</strong>: Uses Foundation Framework with enhanced styling and visual feedback</li>
<li><strong>Advanced chat log parsing</strong>: 
  <ul>
  <li>Handles complex chat formats based on the server's formatting and coloring</li>
  <li>Changeable color for (cellphone) lines</li>
  <li>Censorship options with pixelated or hidden text</li>
  <li>Character name filtering</li>
  <li>Automatic line breaks</li>
  </ul>
</li>
<li><strong>Performance Optimizations</strong>: 
  <ul>
  <li>Debounced input handling for smoother experience</li>
  <li>Optimized image generation</li>
  <li>Loading indicators for long operations</li>
  </ul>
</li>
<li><strong>Enhanced User Experience</strong>: 
  <ul>
  <li>Keyboard shortcuts (Ctrl+S to save, Ctrl+B to toggle background)</li>
  <li>Visual feedback on interactions</li>
  <li>Improved tooltips and help information</li>
  </ul>
</li>
<li><strong>Customization Options</strong>: 
  <ul>
  <li>Switchable black background that wraps around the lines</li>
  <li>Adjustable font size and line length</li>
  <li>Toggle for high resolution images (2x scale)</li>
  </ul>
</li>
</ul>
<h2 id="usage">Usage</h2>
<ol>
<li>Paste your chat log into the text area</li>
<li>Adjust font size and line length as needed</li>
<li>Use the character name filter to highlight specific conversations</li>
<li>Toggle background on/off as desired</li>
<li>Click "Download" to save the image</li>
</ol>
<h3 id="special-formatting">Special Formatting</h3>
<ul>
<li>Start a line with `!` to color it yellow (for phone conversations)</li>
<li>Wrap text with `÷` to censor it (click the ÷ button to copy the character)</li>
<li>Use the "Toggle Censorship Style" button to switch between pixelated and hidden censorship</li>
</ul>
